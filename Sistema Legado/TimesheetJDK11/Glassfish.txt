	Configuration > server-config > Security > Realms menu option, and click the New button.
	JDBCAuth for this example. Next, for the class name, choose com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
	Realm Name: JDBCAuth
	JAAS jdbcRealm
	JNDI jdbc/OracleConnASDB
	User Table COM_ACES_USUARIO
	User Name Column CDG_USUR
	Password Column SENHA
	Group Table COM_ACES_GRUPOS
	Group Name Column CDG_USUR
	Group Name Column: DESCRICAO
	Digest Algorithm: MD5
	Password Encryption Algorithm: MD5
