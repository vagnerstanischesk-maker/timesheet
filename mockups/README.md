# Mockups das telas — Triscal Apontamento de Horas

Protótipos visuais (alta fidelidade, dados fictícios) das telas principais do novo sistema.
Abra cada arquivo `.html` no navegador. Suportam modo claro e escuro automaticamente.

| Arquivo | Tela |
|---------|------|
| 01_login.html | Login / SSO (Entra ID via Keycloak) |
| 02_lancamento_dia.html | Lançamento do dia (mobile, com cronômetro) |
| 03_log_quinzena.html | Log da quinzena (grade dias × tarefas) |
| 04_aprovacao.html | Aprovação de logs (gestor) |
| 05_acompanhamento_projeto.html | Acompanhamento de projeto / fechamento quinzenal |
| 06_dashboard.html | Dashboard gerencial |
| 07_cadastros.html | Cadastros (master-detail, ex.: projetos) |

> São mockups para alinhamento visual; a implementação final usará React + MUI/Mantine,
> conforme `ARQUITETURA_TELAS_COMPONENTES.md`.
